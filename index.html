About This Project

This is a mad lib story generator that I made for my web design class. It uses JavaScript to collect words from the user and then puts them into a story.

How It Works

When you load the page, it will ask you for different types of words like nouns, verbs, and adjectives. Then it uses those words to create a unique story every time.

Made with HTML, CSS, JavaScript and Bootstrap
